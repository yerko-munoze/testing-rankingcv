Ingeniero de Software Backend
Buscamos un/a Ingeniero de Software Backend con sólidos conocimientos en lenguajes de programación como Java, Python o Node.js, 
y experiencia en el desarrollo de APIs RESTful y servicios web. Se valorará experiencia con bases de datos SQL y NoSQL, y 
familiaridad con arquitecturas de microservicios y contenedores como Docker.

Palabras clave: Ingeniero de Software Backend, Java, Python, Node.js, APIs RESTful, servicios web, SQL, NoSQL, 
microservicios, Docker, Kubernetes, bases de datos, desarrollo backend, programación, experiencia.

DevOps Engineer
Se busca DevOps Engineer con 4 años de experiencia en automatización de infraestructuras y gestión de entornos de desarrollo. 
El candidato ideal deberá tener experiencia en herramientas como Docker y Kubernetes, así como conocimientos en CI/CD y 
administración de servidores. Se valorará experiencia en entornos Cloud como AWS o Azure y habilidades en scripting con Python o 
Bash.

Palabras clave: DevOps, automatización, infraestructura, Docker, Kubernetes, CI/CD, servidores, AWS, Azure, Python, 
Bash, scripting, Cloud, gestión de entornos, administración de sistemas, experiencia.

Analista Cloud
Estamos en búsqueda Analista Cloud con experiencia en gestión de proyectos para 
cliente del rubro de seguros.
Colaborar en la implementación y configuración de entornos Cloud, preferentemente en Azure y/o AWS.
Apoyar en la gestión y seguimiento de proyectos utilizando herramientas como Project, Jira y Kanban.
Aplicar metodologías ágiles y de gestión (Agile, Lean, ITIL, PMBOK) en las tareas diarias para asegurar el cumplimiento de los objetivos del proyecto.
Monitorear y optimizar el rendimiento de las soluciones Cloud.
Coordinarse con equipos técnicos y de proyecto para asegurar la integración efectiva de nuevas soluciones.

Palabras clave: Analista Cloud, gestión de proyectos, Azure, AWS, Project, Jira, Kanban, metodologías ágiles, Agile, Lean, 
ITIL, PMBOK, entornos Cloud, eficiencia operativa, seguros, implementación, configuración, optimización, integración,
inglés avanzado.

Científico de Datos

Palabras clave: Científico de Datos, data science, machine learning, big data, Python, R, Hadoop, Spark, análisis estadístico,

seguridad de la información, experiencia.
